🌑 Jujutsu RPG - Terminal Edition
Este é o meu primeiro projeto de programação! Um RPG de texto desenvolvido em Python onde você enfrenta a temível Entity.

🕹️ Como Jogar
O jogo funciona totalmente pelo terminal. Você terá que explorar salas, coletar itens e decidir se aceita poderes amaldiçoados ou se luta como um humano.

Personagens Disponíveis:
Gojo Satoru: Desbloqueado ao aceitar a troca no símbolo. Possui o Vazio Roxo e a Expansão de Domínio.

Itadori Yuji: Desbloqueado através do "Despertar" ao acertar um Kokusen sendo humano. Possui o Punho Divergente e o Corte Mundial.

🛠️ Mecânicas Principais
Sistema de Energia Amaldiçoada (EA): Cada golpe gasta energia, forçando você a gerenciar seus recursos.

Combate Dinâmico: O Boss muda de vida e dano se você despertar um poder maior.

Easter Egg: O ritual do Roxo 200% é ativado se você tiver os pergaminhos azul e vermelho e usar os ataques na ordem certa.

👨‍💻 O que eu aprendi
Neste projeto, pratiquei:

Estruturas de Repetição: Uso intenso de while para manter o jogo rodando.

Lógica Condicional: if, elif e else para criar caminhos diferentes na história.

Gestão de Variáveis: Controle de vida, energia e inventário.

Debug: Resolver erros de identação e conflitos de variáveis.
